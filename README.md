**Sukun (سکن) - Community Mental Health Initiative**

**Overview**
Sukun (meaning "Peace") is a technology-driven community initiative designed to bridge the gap between student mental health crises and professional support.

Based on our primary research survey of 78 participants, we discovered a critical "Silence Gap": 74% of students feel uncomfortable asking for help due to social stigma and privacy concerns. Sukun solves this by providing a high-anonymity, zero-footprint digital portal.

**Key Features**

Anonymous Identity Masking: Uses Java-based UUID logic to generate randomized session IDs, ensuring no personal data is ever logged.

Live Anonymous Chat: A real-time peer support feed where students can encourage one another without fear of judgment.

Sukun Connect: A bridging tool to match students with trained peer mentors anonymously.

Tea & Talk Scheduler: Integration with physical community meetups to move digital support into real-world civic action.

Emergency Directory: One-tap access to verified helplines like 1166 and Umang Pakistan.

**Technical Architecture**

Backend Logic: Developed in Java (Swing/Logic) focusing on secure session handling and unique ID generation.

Frontend Prototype: Built as a Progressive Web App (PWA) using HTML5, Tailwind CSS, and JavaScript for instant access via QR codes.

Deployment: Optimized for mobile-first browser access to avoid the privacy footprint of a native app store download.

**Research Findings**

Our project was driven by data-centric community mapping:

Frequency: 62% of respondents experience stress 3+ times a week.

Barriers: Stigma and "fear of being a burden" were identified as the primary reasons for the 74% help-seeking discomfort.

Coping: Most students currently cope through social withdrawal—Sukun aims to replace withdrawal with Anonymous Engagement.

**How to Run**

Web Version: Open index.html in any modern browser (Chrome/Safari) to view the interactive PWA prototype.

Java Version: Import SukunApp.java into Eclipse or any Java IDE and run as a Java Application to view the desktop management tool.
